# Chaos Clicker

A chaotic Python clicker game where the button runs away, popup windows appear, fake error messages show up, and the game gets mad if you win too hard.

This is a harmless joke game made with Python and Tkinter.

## Features

- Chaotic clicker gameplay
- Moving click button
- Popup windows
- Fake error messages
- Upgrade system
- Coin system
- Task system
- Pause screen
- Start screen with clickable words
- Dev tool for testing levels and features
- Secret hard level 10 ending
- Optional Windows `.exe` build with PyInstaller

## Important Safety Note

This game is harmless.

It does **not**:

- steal data
- delete files
- modify system settings
- install anything
- run in the background
- collect passwords
- connect to the internet

The fake errors and popup windows are only part of the joke gameplay.

## Antivirus / VirusTotal Notice

Some antivirus tools may flag the `.exe` version as suspicious because it is built with PyInstaller and because the game opens popup windows and fake error messages.

This can happen with small Python games packed as `.exe` files.

If you do not trust the `.exe`, you can run the Python source file directly instead.
