# chaos-clicker
A harmless joke clicker game with upgrades, popup windows, fake error messages, a dev tool, and a chaotic secret ending.
⚠️ Hinweis:
Das ist ein harmloses Joke-Clicker-Spiel mit Python und PyInstaller.
Es öffnet Popup-Fenster und zeigt Fake-Fehlermeldungen als Teil des Spiels.
Es stiehlt keine Daten, beschädigt keine Dateien, installiert nichts und verändert das System nicht.

Manche Antivirus-Programme markieren PyInstaller-EXE-Dateien fälschlicherweise als verdächtig.
Wenn du der EXE nicht vertraust, kannst du direkt die Python-Datei ausführen.
